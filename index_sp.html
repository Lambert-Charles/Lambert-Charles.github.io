MD5 algorithm

Hash your passwords

For practice's (and fun's) sake, I wrote my own MD5 hash generator in java. The MD5 message-digest algorithm is a widely used hash function producing a 128-bit hash value. It was designed by Ronald Rivest in 1991 to replace an earlier hash function MD4, and was specified in 1992 as RFC 1321. More information on MD5. I built this project with Maven and it comes along with Junit tests.

  • Language: Java
  • Date: August 2023
